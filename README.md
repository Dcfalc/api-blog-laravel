# Blog - Api
Small back project of a blog, using PHP > Laravel 8.12.

## Project setup
```
composer install
```

Create and configure .env file (database)
```
php artisan migrate
```

```
php artisan db:seed --class=ArticleSeeder
```

### Rum development server
```
php artisan serve
```
